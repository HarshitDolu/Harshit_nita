CipherX

This was a project I started back in my 5th semester when I first learned about cryptograph. The purpose for this project was to get familiar with Django.

This website solves numerical problems of cryptography, especially classical ciphers.

User can learn about cryptography, and can encrypt/decrypt the problems.

User can give feedback regarding it's functioning. This website is hosted on Heroku platform.

I worked on this system for 4 days with my friend.

Technologies:
  • - Python
  • - Django
  • - Postgres
  • - Google Maps API
